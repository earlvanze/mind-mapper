# Mind Mapp — Overview

Mind Mapp is a lightweight, keyboard‑first mind map app focused on speed and clarity. It runs fully in the browser with localStorage persistence and export options.

## Highlights
- Fast node creation (Enter/Tab)
- Keyboard navigation + search
- Autosave, JSON/PNG export
- Offline‑friendly (no backend)

See README.md for setup and RUN.md for shortcuts.

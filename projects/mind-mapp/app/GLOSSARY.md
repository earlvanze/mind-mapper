# Mind Mapp — Glossary

- **Node**: A single text item in the mind map.
- **Root**: The central node (cannot be deleted).
- **Sibling**: A node at the same level (created with Enter).
- **Child**: A node below the current node (created with Tab).
- **Promote**: Move a node up one level (Shift+Tab).
- **Focus**: The currently selected node (keyboard target).
- **Edit mode**: When node text is editable (E or double‑click).
- **Autosave**: Debounced localStorage save after edits.
- **Fit to view**: Zoom/pan to show all nodes (F or toolbar).
- **Reset view**: Reset pan/zoom to default (double‑click background).

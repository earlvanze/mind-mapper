# Mind Mapp — Keybindings (v0.12)

## Node Creation & Editing
| Key | Action |
|-----|--------|
| Enter | Create sibling node (after current) |
| Tab | Create child node |
| Shift+Tab | Promote node (move to parent's level) |
| Delete/Backspace | Delete selected node(s) |
| E / Double-click | Edit node text |
| Cmd/Ctrl+B | Bold text |
| Cmd/Ctrl+I | Italic text |

## Selection
| Key | Action |
|-----|--------|
| Cmd/Ctrl+click | Add/remove node from selection |
| Cmd/Ctrl+A | Select all nodes |
| Alt+I | Invert selection |
| Alt+S | Select siblings |
| Alt+C | Select children |
| Alt+L | Select leaves in subtree |
| Alt+U | Select ancestors |
| Alt+T | Reduce to top-level nodes |
| Alt+G | Select nodes at depth |
| Alt+X | Clear selection extras (keep focus) |
| Alt+N | Expand to parents + children |
| Alt+B | Select focused subtree |
| Alt+P | Select parent |
| Alt+Shift+X | Align selected to focused X |
| Alt+Shift+Y | Align selected to focused Y |
| Alt+Shift+H | Distribute horizontally |
| Alt+Shift+V | Distribute vertically |
| Alt+Shift+R | Layout selected in row |
| Alt+Shift+D | Layout selected in column |
| Alt+Shift+G | Snap to 20px grid |
| Alt+Shift+M | Mirror across focused X |
| Alt+Shift+W | Mirror across focused Y |

## Navigation
| Key | Action |
|-----|--------|
| Arrow keys | Move focus |
| Cmd/Ctrl+K | Search & jump |
| Shift+H/J | Previous/next sibling |
| Shift+N | First child |
| Shift+P | Parent |
| Shift+L | First leaf |
| Shift+K | Last leaf |
| Shift+,/. | Previous/next leaf |
| R | Jump to root |
| Shift+R | Forward in focus history |
| Alt+R | Back in focus history |
| Alt+Shift+Home | Oldest focus history |
| Alt+Shift+End | Newest focus history |

## View & Layout
| Key | Action |
|-----|--------|
| Space+Drag | Pan canvas |
| Cmd/Ctrl+Wheel | Zoom |
| = / + | Zoom in |
| - | Zoom out |
| 0 | Reset pan/zoom |
| F | Fit to view |
| Alt+F | Fit selected nodes |
| Alt+Shift+F | Fit focused subtree |
| C | Center focused node |
| Shift+C | Center root |
| L | Cycle auto-layout (tree → radial → force) |
| Shift+G | Toggle grid overlay |
| Shift+M | Toggle mini-map |
| Shift+A | Toggle advanced toolbar |

## Tags
| Key | Action |
|-----|--------|
| Cmd/Ctrl+T | Open tag picker dialog |
| Cmd/Ctrl+Shift+T | Open tag picker (same) |
| Cmd/Ctrl+Shift+F | Toggle tag filter panel |
| Cmd/Ctrl+1-7 | Apply color preset |

## Copy & Export
| Key | Action |
|-----|--------|
| Cmd/Ctrl+D | Duplicate selected |
| Cmd/Ctrl+C | Copy selected node text |
| Cmd/Ctrl+Shift+L | Copy subtree outline |
| Alt+Shift+P | Copy focused node path |
| Cmd/Ctrl+S | Export JSON |
| Cmd/Ctrl+Shift+S | Export PNG |
| Cmd/Ctrl+Shift+M | Export Markdown |

## Undo/Redo
| Key | Action |
|-----|--------|
| Cmd/Ctrl+Z | Undo |
| Cmd/Ctrl+Shift+Z / Y | Redo |

## Help
| Key | Action |
|-----|--------|
| ? / Cmd/Ctrl+/ | Toggle help dialog |
| Esc | Close dialog / clear selection |

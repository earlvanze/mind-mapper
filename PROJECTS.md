# PROJECTS.md

## Mind Mapp (Mind Map App)
**Status:** Active
**Owner:** Earl Co
**Goal:** Build an MVP mind-mapping app optimized for speed, clarity, and execution workflows.

### MVP Scope (v0.1)
- Create/edit/delete nodes
- Drag to reposition
- Add child node (Enter/Tab)
- Quick search (cmd/ctrl+k)
- Export: PNG + JSON
- Autosave local (browser storage)

### Non-goals (for MVP)
- Real-time collaboration
- Accounts/auth
- Templates marketplace

### Risks / Unknowns
- Best UX for fast node creation on mobile
- Export quality for large maps

### Next Actions
- Draft product spec + UX flows
- Define data model + storage schema
- Decide stack (likely React + Canvas/SVG)
